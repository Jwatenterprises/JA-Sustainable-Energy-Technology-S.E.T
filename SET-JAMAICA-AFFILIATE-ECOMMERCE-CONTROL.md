## Minimum Proof Before Publishing a Checkout Product

| Requirement | Pass condition |
| --- | --- |
| Jamaica shipping | Supplier confirms direct Jamaica delivery or a reliable forwarding path with cost and timeline. |
| Warranty | Written warranty period and claim process. |
| Returns/replacements | Written process for damaged, defective, delayed, or wrong items. |
| Safety/compatibility | Voltage, plug, battery chemistry, waterproof rating, and certification reviewed where relevant. |
| Sample | Sample ordered/tested or a local equivalent inspected. |
| Product page | Clear delivery caveat, warranty wording, return wording, and no unsupported performance claims. |
